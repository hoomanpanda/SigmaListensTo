## SigmaListensTo

Make le meme of Sigma Patrick Bateman listening to the sigma music of your choice.

## Build 

1. Requirements 
    * pyinstaller
            ```
            pip install pyinstaller
            ```

2. Clone this repo
    ```
    git clone https://github.com/hoomanpanda/SigmaListensTo.git
    ```

3. Build 
    ```
    pyinstaller slt.spec
    ```
